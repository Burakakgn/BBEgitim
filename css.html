CSS Nedir ve Nasıl Çalışır

CSS dediğimiz şey aslında Cascading Style Sheets yani Basamaklı Stil Şablonları anlamına gelir. HTML sayfan var diyelim—o sayfa tıpkı çıplak bir manken gibi. İşte CSS, bu mankene ceket giydiren, kravat takan, ayakkabı seçen stilisttir kral.

Ne İşe Yarar?

✅ Yazının rengini değiştirir. ✅ Arka planını boyar. ✅ Kutular çizer, köşelerini yuvarlar. ✅ Animasyon yapar, hover efekti verir. ✅Ekrana göre şekil değiştirir (Responsive).

Yani CSS, bir web sitesinin ruhunu, görünüşünü, havasını belirler. HTML sadece iskelet—CSS ise o iskeletin görünüşüdür

CSS'i Sayfaya Nasıl Ekleriz?

1. Satır İçi(İnline) Tavsiye etmem. Acemiler için <p style="color:red;"> Bu kırmızı paragraf </p> 2. Sayfa İçi(İnternal) < head > içindeki <style> p { color: blue; } </style>
